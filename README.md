#🏉 Proyecto CLUB TABORIN RUGBY - Sitio Web Oficial
Este proyecto es el sitio web del **Club Taborín Rugby** de Córdoba. Está diseñado para ofrecer una experiencia fluida e informativa a los socios, jugadores y fanáticos del club, permitiendo conocer las últimas novedades, categorías e historia de la institución.

## ✨ Características Principales (Features)

- **Diseño 100% Responsivo:** Adaptado para una navegación óptima en smartphones, tablets y computadoras gracias al sistema de grillas de Bootstrap.
- **Componentes Interactivos (Modals):** Ventanas emergentes dinámicas para mostrar información sobre suscripción, recuperación de contraseña y fotos en tamaño completo, sin recargar la página.
- **Blog del Club:** Espacio dedicado a las últimas noticias relacionadas con el equipo, con la posibilidad de dejar comentarios.
- **Galería de Imágenes del Club:** Espacios dedicados a las categorías infantiles, juveniles, plantel superior.
- **Formulario de Login:** Formulario para que los socios del club puedan logearse con su cuenta, con un modal con los pasos para recuperar la contraseña.
- **Página de Contacto:** Formulario de contacto para poder realizar consultas, y un mapa con la ubicación del Club.
- **Página sobre Nosotros:** Foto y nombre de los creadores de ésta páginaweb.

## 🛠️ Tecnologías Utilizadas

Las herramientas elegidas para el desarrollo del sitio garantizan una estructura limpia, estilos modernos y componentes interactivos rápidos:

- **HTML5** - Estructura semántica del contenido.
- **CSS3** - Estilos personalizados y ajustes de diseño específicos del club.
- **Bootstrap 5** - Framework para el diseño responsivo y componentes de interfaz.
- **JavaScript (Bootstrap Scripts)** - Para dar vida a los Modals, Alerts y la barra de navegación colapsable.
